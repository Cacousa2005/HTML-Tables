Table Properties

Table identifiers:
-> Table Header - th --- Defines a header cell. First row. -> Table Footer - tfoot --- Defines the footer cell. Last row. -> Table Body - tbody --- Defines the table without the header and the footer -> Table Row - tr --- Defines a row in an HTML table -> Table Data - td --- Defines the content in a cell

HTML Table - Colspan

To make a cell span two or more columns, use the colspan attribute:

Name Age Language
Ricardo Ferreira 59 Portuguese
Eva Johnson 57 English
Simone Ferrer 45 French


HTML Table - Rowspan

To make a cell span over multiple rows, use the rowspan attribute:

Name Ricardo Ferreira
Phone 555-1234
555-8745
City Porto Alegre


HTML Table - Colgroup

The 'colgroup' tag specifies a group of one or more columns in a table for formatting. It is useful for applying styles to entire columns.
As you can see, it always counts from Left-to-Right
